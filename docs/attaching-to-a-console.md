# <a name="attaching-to-a-console"></a>附加至主控台


進程可以使用 [**AttachConsole**](attachconsole.md) 函式來附加至主控台。 進程可以附加至一個主控台。

主控台可以有許多附加的進程。 若要取出附加至主控台的進程清單，請呼叫 [**GetConsoleProcessList**](getconsoleprocesslist.md) 函式。
