> [!TIP]
> 不建議使用此 API，也不會有對等的 **[虛擬終端](../console-virtual-terminal-sequences.md)** 機。 這種決策刻意將 Windows 平臺與其他作業系統對齊。 此狀態只會與本機使用者、會話和許可權內容相關。 使用此 API 時，透過跨平臺公用程式和傳輸（如 SSH）的應用程式遠端執行可能無法如預期般運作。
