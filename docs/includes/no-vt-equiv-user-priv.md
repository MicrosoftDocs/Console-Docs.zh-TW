> [!TIP]
> 不建議使用此 API，也不會有對等的 **[虛擬終端](../console-virtual-terminal-sequences.md)** 機。 這項決策刻意將 Windows 平臺與其他作業系統保持一致，讓使用者可以完全掌控此簡報選項。 使用此 API 時，透過跨平臺公用程式和傳輸（如 SSH）的應用程式遠端執行可能無法如預期般運作。
